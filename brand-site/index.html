Use The
Master Files.

  • +Keep clear space equal to the emblem height — labelled x — on every side.
  • +Sit the mark on quiet surfaces, at full opacity, in one of the four approved pairings.

Discipline

Surfaces are cloud or ink. Don't introduce new brand hues for surfaces, charts, or components — neutrals build the page; green makes the point. The previous mint-led dark system is retired, and dark mode is eliminated — ink is a local accent, never a theme. Light, always.

Type In Use

One Family.
Two Optical Sizes.

Display · 900 BlackRevenue infrastructure
Display · 700 BoldFor the modern web
Display · 600 SemiBoldThe operating layer for independent web businesses
Text · 400 RegularSay what you mean, plainly and with confidence.
Mono · MediumEPMV · ezID · JS snippet · #70A92A

Headings — Inter Display, −5% letter spacing, 125% line spacing. Title Case, closed with a period.

Body — Inter Text, −3% letter spacing, 150% line spacing, sentence case.

Nothing is set in all caps. Hierarchy comes from scale and spacing — not caps, not bold runs.

Reddit Mono is reserved for code, file paths, hex, and exact values. Never decoration.

The wordmark is never typeset. It is a file, placed.
